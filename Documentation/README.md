# Procedural Mesh Generator

**Create 3D Meshes Procedurally in Unity**

Generate rocks, crystals, and organic shapes with code-based mesh generation.

## Features

- **6 Mesh Types** - Rock, Crystal, Sphere, Tube, Torus, Terrain
- **Noise Displacement** - Organic-looking deformation
- **Random Seed** - Reproducible results
- **Variations** - Generate multiple versions at once
- **Export Options** - Save as prefab or mesh asset
- **LOD Support** - Generate level-of-detail meshes

## Installation

1. Import this package into your Unity project
2. Go to **Tools > Procedural Mesh Generator**

## Usage

1. Select mesh type
2. Adjust detail level and size
3. Set noise parameters for organic shapes
4. Click "GENERATE MESH"
5. Save as prefab or export as asset

## Requirements

- Unity 2021.3 or later
- Universal Render Pipeline (URP)
